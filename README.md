# Nexus-Corp KBDSS MVP

Proyecto estático listo para desplegar en Vercel.

## Archivos
- `index.html`: aplicación completa.
- `vercel.json`: configuración mínima para Vercel.
- `package.json`: scripts básicos.

## Despliegue rápido
1. Sube esta carpeta a GitHub.
2. En Vercel, selecciona **Add New > Project**.
3. Importa el repositorio.
4. Framework Preset: **Other**.
5. Build Command: vacío.
6. Output Directory: vacío o `.`.
7. Deploy.
